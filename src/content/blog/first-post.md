Really excited to come back to IME-USP after some years. During graduation, I didn't take this course, so I came with an open mind (and a little bit afraid of C).

I had to skip the first class as I was travelling, so I did the [first tutorial](https://flusp.ime.usp.br/kernel/qemu-libvirt-setup/) during the second class.

Since I already had previous experience using VMs and Linux, it was easy (and nostalgic) to follow the well-written and clear tutorial. A couple observations:

- I had to install `ssh-server` inside the VM.
- I had to change the version of Debian to download, since the tutorial used a daily version that isn't available anymore.
- The longest part was downloading the images using _eduroam_ Wi-Fi. 

By the end of tutorial, I had a VM running a Debian image 👍.

And that's it for the first post. Let's see how the next ones go :)
