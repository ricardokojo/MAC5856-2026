This post is about my experience following [tutorial 4](https://flusp.ime.usp.br/kernel/char-drivers-intro/).

This one was a little bit different than the others. The others were hands on from the start, while this had a lot of content before actually doing anything, which by itself is not a problem.

The content was easier to understand because we had an introduction to it during last class. Without it, I feel like it'd be much harder to grasp.

When it came to creating the character driver and rebuilding the kernel, it was fairly easy. Just had to go back to the previous tutorials to get all the commands. It took around 12 minutes to build the kernel again and all the other steps ran without issues.

However, since this had a lot of premade code, I'm not sure if I'd be able to complete the two proposed exercises without support.
